# Gaia Temple

*A nonprofit spiritual and educational organization supporting regenerative culture and community.*

---

## About This Repository

This repository hosts the source code for the official **Gaia Temple** website.

The site serves as a public-facing platform for education, storytelling, and collaboration around regenerative culture, community initiatives, governance experiments, and integrative systems.

---

## About Gaia Temple

Gaia Temple is a nonprofit spiritual and educational organization that supports regenerative culture through education, community gathering, systems design, and land-based initiatives.

It functions as a connective platform—linking people, projects, ideas, and resources in service of ecological, cultural, and social regeneration.

---

## About the Website

This website is intended as a living digital space for sharing ideas, frameworks, and initiatives connected to Gaia Temple’s mission.

It is not a commercial product, financial platform, or formal governance authority. Participation and engagement are invitational and educational in nature.

---

## Technical Notes

The site is built as a static web project using HTML, CSS, and JavaScript and is hosted via GitHub Pages. This approach preserves visual integrity, accessibility, and independence from proprietary platforms.
